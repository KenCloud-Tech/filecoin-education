Kourse : Learn. Earn. Contribute.

Start with Kourse Data DAOs that can gamify education, educating its user base through engagement with purpose-guided education.

  • Overcome free entertainment using gamified paid-to-learn blockchain engagement mechanics.
  • Utilize "smart-data" NFTs, encapsulating lecture edu video/audio/slide/q&a data with NFT keys.
  • Distribute NFTs to lecturer/attendees, charity, sell in aftermarkets for funding.
  • Lecturer/student have cryptographic keys for resume/CV.
  • Peer-to-peer education is non-fungible.
Manifesto

Data DAOs + Non-Fungible Educational Ecosystems

If peer to peer education is non-fungible, and Data DAOs need educated members, then tokenizing and gamifying the education process could be used as a means to educate and onboard more active Data DAOs participants..
